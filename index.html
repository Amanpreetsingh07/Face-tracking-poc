About

This module uses Face-api.js to utilise lightweight face detection models like Tiny face detector to track face landmarks. The library provides a huge array of co-ordinates that can be used to do custom calculations as per requirement.

This particular model generates landmarks on a 2d plane. However, combining data from tilt and direction, 3d orientation can be computed to some extent

Known issues

  • Bad detection in low light conditions
  • Bad detection when neck is tilted at extreme angles
  • Bad data when face is partially outside the video frame
